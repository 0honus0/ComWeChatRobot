# 描述
PC微信机器人，实现获取通讯录、发送文本、图片、文件、xml文章、分享名片消息  
封装COM接口，可供Python、C#调用
# 可用版本
微信电脑版**3.5.0.46**  
(已非最新版，目前微信更新了3.6版本)
# 编译环境
**Visual Studio 2019**
# 原理
通过逆向PC微信，定位到关键CALL，dll内联汇编调用  
注册32位COM组件，供64位/32位进程外部调用  
# 注册COM
以管理员权限执行以下命令：  
```shell
# 安装
CWeChatRobot.exe /regserver
# 卸载
CWeChatRobot.exe /unregserver
```
